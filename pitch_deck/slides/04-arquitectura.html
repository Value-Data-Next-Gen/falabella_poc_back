Arquitectura

Componentes desacoplados, comunicación HTTP + JWT

FRONTEND

React + Vite

  • TypeScript + TailwindCSS
  • Deck.gl + MapLibre (mapa)
  • Recharts (gráficos)

BACKEND

FastAPI + Python 3.11

  • XGBoost + SHAP (modelo)
  • APScheduler (tick 3s)
  • Live generator SQL
  • Pydantic v2 (schemas)
  • RBAC por empresa

STORAGE

Azure SQL (fpoc)

  • simpli_visits (180k rows)
  • geo_suborders
  • users · empresas
  • vip_clients
  • notifications_log

MENSAJERÍA

Twilio WhatsApp
